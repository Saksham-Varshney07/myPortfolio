# Saksham Varshney - Interactive OS Portfolio 

Welcome to the source code of my personal portfolio! 

Instead of a traditional scrolling website, this portfolio is designed as a highly interactive, MacOS-inspired desktop environment. It features draggable windows, a functional dock, dynamic widgets, and an immersive user experience built from the ground up.

##  Features

- **OS-like Interface:** Draggable windows, a bottom dock, and a status bar (using Framer Motion).
- **Cinematic Project Previews:** Hover over projects to trigger immersive, full-screen video workflow demonstrations.
- **Dynamic Themes:** Custom background themes (like 'Spiderverse' and 'Barcelona') complete with built-in ambient music.
- **Interactive Widgets:** Live GitHub contribution heatmap, a real-time clock, status indicators, and more.
- **Fully Responsive:** Adapts from a complex desktop environment to a clean, native-feeling mobile layout on smaller screens.

##  Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router, React 19)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Language:** TypeScript

##  Getting Started

To run this project locally, simply clone the repository and install the dependencies:

```bash
git clone https://github.com/Saksham-Varshney07/portfolio.git
cd portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
